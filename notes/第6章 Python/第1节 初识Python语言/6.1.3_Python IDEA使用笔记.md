## 快捷键

| 功能           | 快捷键     | 功能     | 快捷键             |
| -------------- | ---------- | -------- | ------------------ |
| 返回上一条命令 | `alt + p`  | 修改字体 | `option`里面       |
| 后退           | `ctrl + z` | 重做     | `ctrl + shift + z` |
| 加缩进         | `ctrl + ]` | 减缩进   | `ctrl + [`         |
| 加注释         | `alt + 3`  | 去注释   | `alt+ 4`           |
