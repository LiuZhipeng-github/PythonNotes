# 软件快捷键记录

---



4. Linux系统

| 功能             | 快捷键                | 功能       | 快捷键       |
| ---------------- | --------------------- | ---------- | ------------ |
| 窗口放大         | `Ctrl + shift + 加号` | 窗口缩小   | `Ctrl + -`   |
| 自动补全         | `Tab`                 | 中英文切换 | `win + 空格` |
| 调用之前写的命令 | `↑`                   | 当前目录   | `.`          |
| 上一级目录       | `..`                  |            |              |
