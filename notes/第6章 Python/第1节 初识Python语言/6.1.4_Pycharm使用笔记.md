## 关闭双击`shift`打开全局搜索

双击`shift` -->  `actions`  -->  `registry`  --> 找到`ide.suppress.double.click.handler`勾选

开启：`ctrl + shift + A`



## 快捷键

| 功能               | 快捷键                  | 功能           | 快捷键           |
| ------------------ | ----------------------- | -------------- | ---------------- |
| 向左缩进           | `shift + tab`           | 多行同时输入   | `Alt + 行`       |
| 自动导包           | `alt + enter`           | 快速复制粘贴   | `ctrl + d`       |
| 代码上下移动       | `ctrl/alt + shift + ↑↓` | 代码快速格式化 | `ctrl + alt + L` |
| 光标到最前和最后面 | `home/end`              |                |                  |
